## What is Relay?

> **Git made code collaboration possible. Relay does the same for the AI working alongside your team.**

Relay is a Claude Code plugin that gives a team **one shared memory** across every teammate's sessions. No one has to change how they work.

Right now, every teammate has a different Claude — blank, unconfigured, and unaware of what the rest of the team has built, decided, or deliberately left unfinished. Relay gives your whole team one shared brain.

When anyone on the team runs Claude Code, Relay:

1. **Watches** the session transcript in the background.
2. **Distills** decisions, rejected paths, and live workarounds into a compact `memory.md`.
3. **Syncs** that memory to every teammate via git.
4. **Injects** it as hidden context at every new `SessionStart` — so any Claude, on any laptop, starts warm.

## How Relay is different

| | What it does for teams |
|---|---|
| Git | Syncs code, not comprehension. |
| `CLAUDE.md` alone | Static; someone has to remember to write it — at the moment they have least time. |
| Claude Projects | Shared docs, but no live session memory or cross-teammate sync. |
| Agent Teams | Multi-agent coordination under one human. Doesn't help N humans with N Claudes. |
| **Relay** | **Continuously distilled, git-synced, auto-injected team memory.** |
